# 时间轴小工具

一个用于管理与展示时间轴事件的 Web 应用。支持本地持久化、导入/导出（CSV/JSON）与导出图片（PNG 透明 / JPG 白底）。

## 功能
- [x] 添加 / 编辑 / 删除时间轴事件
- [x] 搜索与排序（按日期）
- [x] 本地存储（LocalStorage）
- [x] 导入 / 导出 CSV、JSON
- [x] 导出图片（PNG 透明 / JPG 白底）

## 技术栈
- React
- Vite
- Tailwind CSS
- Framer Motion

## 安装
```bash
npm install
```

## 开发
```bash
npm run dev
```

## 构建
```bash
npm run build
```

## 部署
- 直接将 `dist` 目录部署到任意静态托管服务即可。
